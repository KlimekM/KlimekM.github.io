Margins, Borders, and Paddings!

A description of each and how they differ.

November 29, 2015

Margins, borders, and paddings were elements that I had a difficult time wrapping my head around at the start of this week. Everything seemed very self-explanatory based on their title and I figured I would jump into this week's challenges without knowing exactly what they are or how they functioned.

After some trials, errors, and endless research I came across "the box model" that really brought the concept together for me. One thing to keep in mind prior to viewing the box model is that every element in HTML is essentially a box. It does not make a difference if the element is an image, paragraph, header, or any number of other elements, it is a box!

The Box Model:

As you can see in the image above, the box has 4 elements which are content, padding, border, and margin. The content is where the text and images that the creator has inserted all appear, and the border is the outline or border that surrounds the padding and content. The border can be visible and any color that the or width that the creator decides. As seen above the padding lies between the content and the border, it clears an area around the content and if we were to adjust the padding it could visibly stretch the box and it's borders even though it itself is transparent. The margin is the final element that encompasses the previous three. It can not be seen, but it clears a set area around the border that no other elements can intrude upon.

Margins, borders, and paddings can all be adjusted as a whole by inserting margin: "insert value", padding: "insert value", or border: "insert value" . The "insert value" is where the creator or user can set the value of the command to whatever they desire. In addition to changing all of the element, we are able to change each side of the element and make it a different value which can be seen below:

In the image above, the command is margin: and we are changing all 4 sides in a clockwise direction starting from the top. So the top margin would be 1px, the right would be 2px, the bottom would be 3px, and finally the left would be 4px.

Aside from changing all 4 sides of an element we are also able to change each given side and can do so by inserting padding-left: "Insert value" The "insert value" can be the value for the element that we want to change. In this example we are changing the left padding. The left can also be replaced by top, bottom, or right depending on which padding it is that we want to change.

There are many ways to alter and change margins, borders, and paddings. Hopefully I was able to give you a brief introduction into what each element is, how they function, and a few examples of how they can be modified.

Thanks for reading, and happy coding!
